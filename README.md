# Linux Update Script

This repository contains `update.sh`, a simple Bash script to automate system updates on Ubuntu.

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/bitbytebandit/linux-update-script.git
   
2. To run script:
   ./update.sh

**Features**
-Updates the package list.
-Installs available updates.
-Removes unnecessary packages.
